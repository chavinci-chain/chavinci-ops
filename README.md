Install the package via `yarn`:

```sh
yarn add chavinci-ops
```

or via `npm`:

Install the package via `npm`:

```sh
npm install chavinci-ops --save
```

or via `bun`:

Install the package via `bun`:

```sh
bun install chavinci-ops 
```
